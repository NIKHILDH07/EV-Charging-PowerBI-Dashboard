# Electric Vehicle Charging Infrastructure Analysis (Power BI)

## Project Overview

This Power BI dashboard analyzes Electric Vehicle (EV) charging infrastructure usage across cities. The goal is to identify high-demand locations, station utilization, and charging trends.

## Key Metrics

* Total Charging Sessions
* Total Revenue
* Total Energy Consumption

## Visualizations

* EV Charging Demand by City (Map)
* Station Demand Analysis
* Charging Sessions Trend
* KPI Summary Cards

## Tools Used

* Microsoft Power BI
* Excel Dataset
* DAX Measures

## Insights

* Certain cities show higher charging demand
* Station usage varies significantly
* Charging sessions show increasing trend over time
